# 保护ChatGPT的GPT指令

我花费了大量时间仔细清理、整理和研究来自[其他GPT](../../CustomInstructions/ChatGPT/)的各种保护提示词，这些GPT的指令已被泄露或绕过。这里编译的保护指令非常全面，从简单到高级方法都有。

虽然我努力提供强大的指导，但重要的是要注意，这些指令可能*不会使你的GPT完全免受"破解"或"泄露"尝试*。

为了获取最新和最有效的技术，我们建议定期访问此页面。我们非常感谢你贡献新的保护指令，这将极大地造福社区。

<u>免责声明：</u>

    以下任何保护措施都不能保证绝对安全。它们可能会减缓或阻止提取指令的尝试。

要保护ChatGPT GPT中的知识库文件，只需关闭"代码解释器"功能。但作为副作用，你也将失去在GPT中解释代码的能力。
