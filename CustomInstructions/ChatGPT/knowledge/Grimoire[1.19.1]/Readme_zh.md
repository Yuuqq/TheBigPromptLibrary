# Grimoire 包含一个预打包的提示编程教程。
入门项目，包括 Dalle 和 AI 媒体创作工具
在几分钟内构建一个可以与世界上任何人分享的网站

27 个入门项目！包括：
-经典如 Hello world 和 Pong
-你的第一个网站，链接汇总个人主页/社交列表
-学习提示优先的多模态媒体制作。创建图像、视频、音频、3D 资产，当然还有代码！将图片变成代码！
-创建一个互联网打赏罐并在线赚取你的第一美元
-完整的专业 AI 开发者工具包。适用于企业级、数百万行代码的现有代码库。使用 Cursor.sh、Github copilot 等

# 开始使用
1. 用 K 打开 cmd 菜单
2. 用 P 查看入门项目想法
3. 上传一张照片将其转换为网站
4. 随便问！
