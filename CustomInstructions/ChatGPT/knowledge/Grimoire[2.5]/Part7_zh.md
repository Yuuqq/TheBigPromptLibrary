赋值在概念上非常简单。你正在将一个值存储在一个命名的存储空间中：
int a = 2;
这是一个整数变量，名为 a，值为 2

你可以稍后使用存储盒的名称访问该值
print(a); // 打印 2

哦

# 一些语言使用不同的格式
# 和 // 是最常见的。
你也可以使用 /* ... */ 来覆盖大面积区域。
小技巧：几乎每个编码 IDE 都使用 cmd + / 作为热键

回到变量
你可以用它们进行进一步的赋值：

// 从3开始，加上我们之前在 a 中的值2
int b = 3;
b = b + a; // b 现在是5

a 的值，即2，被访问并添加到 b 的值，即3
总和被存储回 b 的空间，替换之前的值3

注意 = 在编程中与数学中不同！这不是等号！
在编程中你使用 == 表示相等。更多内容在我们讲到布尔值时会说。

重要：记住 a 和 b 不是对象

67: 作用域与流程。If、枚举、循环、数组、递归。
通常程序从上到下进行，一行一行
执行每一行，然后进入下一行
但一个只能走一条路的程序真的很无聊

一旦你有办法改变顺序，程序就变得有表现力
实现这一点的机制称为控制流
两种最常见的是分支和循环

这是最简单的分支
如果 a 大于2，b 被设置为20
如果 a 是2或更小，什么都不会发生，程序跳过它

if (a > 2) {
    b = 20;
}

这是另一种写法，显示布尔值，为了更清晰而分开

bool isAIsGreaterThan2 = a > 2;
if (isAIsGreaterThan2) {
    b = 20;
}

如果你想检查某事是否相等，你可以使用 ==
if (a == 2) {
    b = 20;
}

使用 != 检查某事是否不相等
if (a != 2) {
    b = 0;
}


这是一个双向分支：

if(a > 2) {
    b = 20;
}
else {
    b = 4;
}

如果 a 大于2，b 被设置为20
如果 a 是2或更小，b 被设置为4


条件被检查，只有两个不同块中的一个会被执行
你可以把这些变得相当复杂，添加任意多的检查、选项、行或嵌套行

if(a > 2) {
    b = 20;
}
else if (a > 10) {
    b = 50;
}
else if (a > 100) {
    b = 100;
}
else {
    if (a == 1) {
        b = 5;
    } else {
        if (a == 99) {
            b = 99;
        } else {
            b = 6;
        }
    }
}

布尔逻辑是将多个条件组合成一个的方法
这里我们有两个条件，a 大于2，和 c 小于10
如果两者都为真，b 被设置为20

if (a > 2 && c < 10) {
    b = 20;
}

这里我们有两个条件，a 大于2，和 c 小于10
如果任一为真，b 被设置为20

if (a > 2 || c < 10) {
    b = 20;
}

顺便说一下，那些看起来像 E 和 N 的大而可怕的数学数字，大写的 sigma 和大写的 pi，实际上就是 for 循环


sum = 0;
for ( n=0; n<=4; n++ ) {
    sum += 3*n;
}

prod = 1;
for ( n=1; n<=4; n++ ) {
    prod *= 2*n;
}

另一种数据类型是枚举，它们有点像类别或标签
你已经遇到了你的第一个
布尔值！

布尔值只是带有 true 或 false 情况的枚举

enum AnimalType {
    case dog
    case cat
    case bird
}

myAnimal = .dog

if (myAnimal == .dog) {
    print("woof")
} else if (myAnimal == .cat) {
    print("meow")
} else if (myAnimal == .bird) {
    print("tweet")
}

枚举很棒，因为你可以在 switch 语句中使用它们，这是处理稍后添加新情况的好方法

switch myAnimal {
    case .dog:
        print("woof")
    case .cat:
        print("meow")
    case .bird:
        print("tweet")
}

另一种主要类型的控制流是循环

这是一个循环：

int a = 0;
while (a < 10) {
    a++; // a = a + 1 的简写;
}
print(a); // 打印 10

这个循环一遍又一遍地继续，直到条件被改变
小心不要创建任何无限循环！

这是另一个计数到10的循环
这使用常见的 C 风格 for 循环语法。它看起来很疯狂，但实际上很简单

for (int i = 0; i < 10; i++) {
    print("loop number" + i);
}

// int i = 0; 是设置一个索引变量，类似于上面的 while 循环中 int a = 0 做的事情。它在 for 之后，只是为了格式美观，想法是这个数字只会在循环中使用。我们使用索引变量来计算我们经历了多少次循环
// i < 10; 说当 i 小于10时，继续这个循环。10之后，停止循环
