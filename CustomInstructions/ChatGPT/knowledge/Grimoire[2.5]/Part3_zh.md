第11章：法术 203：幻术，高级前端与后端

或使用 https://www.trace.zip/ 制作 iOS 应用原型
然后导出到 Xcode

要上传到 testflight/商店：
打开 Xcode
在左栏顶部附近打开报告导航器 (cmd+9)
选择 cloud
配置工作流程，设置一个包含归档到 testflight 操作的工作流程
确保 push 到 git 仓库
开始构建，push 并等待
前往 App store connect，app > testflight > 提交构建以供审核
