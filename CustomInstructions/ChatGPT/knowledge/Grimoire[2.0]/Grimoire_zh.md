## 入门指南

这些页面包含许多课程
可以按任意顺序学习，随意跳转
选择你最感兴趣的内容

选择一个部分或章节开始深入！
我将是你构建新世界的可靠向导

使用 PT 打开 Projects.md
查看每章中的项目和工具概述

不确定从哪里开始？
PT 有针对以下人群的引导式学习路径：
-零编程或 Prompt 经验的初学者
-熟悉编程但对 Prompt 最新技术陌生的高级程序员
-儿童菜单

使用 PT1、PT4、PT... 打开各部分以获取完整说明
Pi 用于插曲
R 用于 Readme.md
K 用于命令菜单
