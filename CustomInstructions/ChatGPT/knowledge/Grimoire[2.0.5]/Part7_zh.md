这是一个整数变量，名为 a，值为 2

你可以稍后使用存储盒的名称访问该值
print(a); // 打印 2

回到变量
你可以用它们进行进一步的赋值：

// 从 3 开始，加上我们之前的值 2，存在 a 中
int b = 3;
b = b + a; // b 现在是 5

a 的值，即 2，被访问，并加到 b 的值，即 3
总和被存回空间 b，替换之前的值 3

注意 = 在编程中与数学中不一样！这不是等号！

重要：记住 a 和 b 不是对象

67：作用域和流程。If、Enum、循环、数组、递归。
通常程序从上到下，逐行执行
执行每一行，然后移动到下一行
但只能单向执行的程序真的很无聊

一旦你有了改变顺序的方法，程序就变得有表现力
这样做的机制叫做控制流
两种最常见的是分支和循环

这是最简单的分支
如果 a 大于 2，b 被设为 20
如果 a 是 2 或更小，什么都不发生，程序跳过它

if (a > 2) {
    b = 20;
}

这是另一种写法，显示布尔值，分开写以更清晰

bool isAIsGreaterThan2 = a > 2;
if (isAIsGreaterThan2) {
    b = 20;
}

如果你想检查是否相等，可以使用 ==
if (a == 2) {
    b = 20;
}

使用 != 检查是否不相等
if (a != 2) {
    b = 0;
}


这是一个双向分支：

if(a > 2) {
    b = 20;
}
else {
    b = 4;
}

如果 a 大于 2，b 被设为 20
如果 a 是 2 或更小，b 被设为 4


条件被检查，只有两个不同块中的一个会被执行
你可以把这些弄得相当复杂，添加任意多的检查、选项或行或嵌套行

if(a > 2) {
    b = 20;
}
else if (a > 10) {
    b = 50;
}
else if (a > 100) {
    b = 100;
}
else {
    if (a == 1) {
        b = 5;
    } else {
        if (a == 99) {
            b = 99;
        } else {
            b = 6;
        }
    }
}

布尔逻辑是将多个条件组合成一个的方法
这里我们有两个条件，a 大于 2，c 小于 10
如果两个都为真，b 被设为 20

if (a > 2 && c < 10) {
    b = 20;
}

这里我们有两个条件，a 大于 2，c 小于 10
如果任一为真，b 被设为 20

if (a > 2 || c < 10) {
    b = 20;
}

另一种数据类型是枚举，它们有点像类别或标签
你已经遇到了你的第一个
布尔值！

布尔值就是只有 true 或 false 情况的枚举

enum AnimalType {
    case dog
    case cat
    case bird
}

myAnimal = .dog

if (myAnimal == .dog) {
    print("woof")
} else if (myAnimal == .cat) {
    print("meow")
} else if (myAnimal == .bird) {
    print("tweet")
}

枚举很棒，因为你可以在 switch 语句中使用它们，这是处理以后添加新情况的好方法

switch myAnimal {
    case .dog:
        print("woof")
    case .cat:
        print("meow")
    case .bird:
        print("tweet")
}

另一种主要的控制流类型是循环

这是一个循环：

int a = 0;
while (a < 10) {
    a++; // a = a + 1; 的简写
}
print(a); // 打印 10

这个循环一遍又一遍地继续，直到条件改变
小心不要创建任何无限循环！

这是另一个计数到 10 的循环
这使用常见的 c 风格 for 循环语法。它看起来很疯狂，但其实很简单

for (int i = 0; i < 10; i++) {
    print("loop number" + i);
}

// int i = 0; 是设置一个索引变量，类似于上面 while 循环中的 int a = 0。它在 for 之后，只是为了格式化美观，这个数字只会在循环中使用。我们使用索引变量来计算我们经过了多少次循环
// i < 10; 说只要 i 小于 10，就继续这个循环。10 之后，停止循环
